# 🍅 Pomodoro Timer

A beautiful, responsive Pomodoro timer built with React, Vite, and Tailwind CSS.

## 🚀 Live Demo

**[Try it here!](https://pomodoro-timer-iota-bice.vercel.app/)**

## ✨ Features

- ⏱️ 25-minute work sessions
- ☕ 5-minute break sessions
- 🔊 Sound notifications
- 🎨 Modern glassmorphism UI
- 📱 Fully responsive design
- ⚡ Fast and lightweight

## 🛠️ Technologies

- React 18
- Vite
- Tailwind CSS
- Web Audio API

## 🎯 What I Learned

- React hooks (useState, useEffect)
- Component composition
- Props and callbacks
- Responsive design with Tailwind
- Git workflow and branching
- Web Audio API
- Deployment with Vercel

## 🏃‍♂️ Run Locally
```bash
# Clone the repo
git clone https://github.com/YOUR-USERNAME/pomodoro-timer.git

# Install dependencies
cd pomodoro-timer
npm install

# Run dev server
npm run dev
```

## 📦 Build
```bash
npm run build
```
